NAME

		wc -word count

SYNOPSIS

		wc [OPTION] [FILE]

DESCRIPTION

		Count number of lines/words/characters in a file

OPTIONS

        -l
            count number of lines in file
        -w  
            count number of words in file
        -m
            count number of characters in file

EXAMPLE

        wc -w words.txt 
            counts the number of words in words.txt
